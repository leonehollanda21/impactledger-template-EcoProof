# 🌿 EcoProof — Frontend

Bem-vindo ao módulo de Frontend do **EcoProof**, uma solução desenvolvida para o desafio **ImpactLedger** no **Hackathon Web3 RESTIC 29**.

## 💻 Sobre o Módulo
O Frontend é a interface principal da plataforma, construída para proporcionar uma experiência fluida e acessível a três perfis: **Cidadão, Instituto (ONG) e Administrador**. É através dessa interface que os usuários podem registrar ações socioambientais (como coletas de lixo, adoção de áreas verdes e educação ambiental), fazer denúncias e acompanhar o painel de certificações e recompensas que serão auditadas na blockchain.

## 🛠 Tecnologias Utilizadas
- **Vue 3**: Framework progressivo adotado pela sua reatividade e componentização.
- **Vite**: Ferramenta de build de altíssima performance para desenvolvimento moderno.
- **Node.js & NPM**: Ecossistema de gerenciamento de pacotes e servidor de desenvolvimento.

## 🚀 Como Executar Localmente

### Pré-requisitos
- Node.js (versão 18 ou superior)
- Conexão com o Middleware (Backend) rodando localmente.

### Passo a Passo

1. **Acesse a pasta do frontend:**
   ```bash
   cd EcoProof_frontend
   ```

2. **Instale as dependências do projeto:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   Abra seu navegador e acesse a URL indicada no terminal (geralmente `http://localhost:5173`).
